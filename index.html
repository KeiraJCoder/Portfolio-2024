About Me

Hi, I'm Keira Jarvis, a dedicated software developer and instructor with a passion for creating innovative web applications and empowering others to excel in technology. With expertise in front-end and back-end development, I specialise in building intuitive, user-friendly websites and applications that blend creativity with robust technical functionality.

My portfolio reflects a strong foundation in technologies like HTML, CSS, JavaScript, React, Node.js, and API integrations, with a growing focus on security and best practices in development.

My recent recognition includes being shortlisted for the Tech Diversity, Equity, and Inclusion Advocate of the Year 2023 award for my role as an Equality, Diversity, and Inclusion Champion at Code Nation.

When I'm not coding or teaching, you’ll find me exploring new technologies, contributing to open-source projects, or expanding my knowledge in cybersecurity.

Let’s connect! You can view my projects on GitHub or reach out via LinkedIn.
